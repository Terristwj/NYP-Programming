Windows Forms Project to convert .mdf (SQL database) to xls (Microsoft Excel)

Guide to start creating:
1.	Open Visual Studio
2.	Create Windows Forms (Ver 4.6)
3.	Connect SQLite references
4.	Connect Excel references
5.	Start development
6.	Refer to the links aboves for development

Guide to open:
Alt 1.	Open Visual Studio
Alt 2.	Run .exe file inside project > bin > debug

Guide to use:
1.	Connect SQLite database
2.	Test connection (Optional)
3.	Generate Excel
