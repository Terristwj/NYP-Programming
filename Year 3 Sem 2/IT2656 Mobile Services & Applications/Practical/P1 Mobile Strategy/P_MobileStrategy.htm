1         Ionic Command Line Interface (CLI)

1.1        Create Ionic App

1.    Create a folder IonicApps using Finder on Mac or FIle Explorer on Windows.


If you are using the school’s computer, you are highly encouraged to create the folder in D:\.

2.      Open the program Visual Studio Code.

3.      Select View > Terminal.

4.    In the Terminal, go to the IonicApps folder using the  cd  command.

cd IonicApps

 

Example 1:

D:

cd D:\IonicApps

 

 

Example 2:

Z:

cd Z:\Users\cordelial\Desktop\IonicApps

 

           

5.    Type the following command to create an Ionic app called skippyQ.

ionic start skippyQ sidemenu

 


ionic start

skippyQ                          Create a folder called skippyQ with the Ionic structure

sidemenu                        Uses the sidemenu starter template

 

 

6.    Install the free Ionic Appflow SDK and connect your app?

n

 

This creates a folder skippyQ.


All subsequent Ionic/Cordova commands must be carried out within the created skippyQ folder.

 

Explore the folder skippyQ by using Finder on Mac or File Explorer on Windows. Note that there is an auto-generated src folder.
 

1.2        Preview Ionic App 

1.    Go to the skippyQ folder using the  cd  command.

cd skippyQ

 

Test the app in your desktop browser by typing the following command.

ionic serve

 

You will see the default starter template appear in the web browser http://localhost:8100/

 

 


Ionic project Directory

If you encounter the following error: Sorry! ionic serve can only be run in an Ionic project directory, make sure to go to your newly created project by typing

cd skippyQ

 

2.       It is recommended to use Google Chrome to open the app.
If not already done, set Google Chrome as your default browser.
Bring up Search . In the Search bar, type default browser. Select Choose a default web browser.


Set Google Chrome as your default browser.

 


DO NOT DELETE the skippyQ folder on your computer.
Keep the files in a safe space. Backup the files to your USB drive or Google Drive.

2         Branding

2.1      App Name and Icon

1.    Download the zip file from Blackboard.

2.    Extract to src > assets folder.

3.    Open the file src > index.html.
Find <title> and change the text.
Find <link> and change the href.

index.html

<!DOCTYPE html>

<html lang="en">

 

<head>

  <meta charset="utf-8" />

  <title>Skippy Q</title>

  ...

  <link rel="icon" type="image/png" href="assets/icon/icon.png" />

  ...

</head>

 

...

 

4.    Press Ctrl+S  to save the file.

5.    Open the Terminal.
Press Ctrl+C  to cancel the live preview if it is still running.
Re-run the app in your desktop browser by typing the following command.

ionic serve


Verify that the new Mobile Web name and icon are shown in the web browser.

 

 

 In subsequent practical, we will learn how to change the App name and icon for Android and iOS.

 

2.2      Enable Auto Save

For convenience, you may enable Auto Save for Visual Studio Code.

1.    In Visual Studio Code, select File > Auto Save.

 

2.    You should now see a tick beside Auto Save.

 

2.3      Create a Theme

1.    Go to https://ionicframework.com/docs/theming/themes

2.    Enter Background and Text color with your favorite colors.
You may get color hex code from https://htmlcolorcodes.com/
Copy the CSS Variables.

3.    In Visual Studio Code, open the file src > theme > variables.scss.
Paste inside :root.


There should only be one :root.

DO NOT have multiple :root.

DO NOT nest your :root.

 

4.    Open https://ionicframework.com/docs/theming/color-generator

5.    Type in your favourite color hex codes.

6.    In Visual Studio Code, open the file src > theme > variables.scss.
Paste inside :root.


There should only be one :root.

DO NOT have multiple :root.

DO NOT nest your :root.

 

2.4      Customize the App

1.    Open the file src > app > home > home.page.html.
Find <ion-toolbar> and change the color theme.
Find <ion-content> and change the banner and text too.
You may also add a paragraph inside <ion-card-content> to customize your app.

home.page.html

<ion-header>

  <ion-toolbar color="primary"  >

    ...

  </ion-toolbar>

</ion-header>

 

<ion-content>

  <ion-card class="welcome-card">

    <img src="/assets/banner.jpg" alt="" />

    <ion-card-header>

      <ion-card-subtitle>Skip the Queue</ion-card-subtitle>

      <ion-card-title>Skippy Q</ion-card-title>

    </ion-card-header>

...

If you have not enabled Auto Save, please Save the file using Ctrl+S.

2.    Visual Studio Code supports Live Preview. If you have not closed the browser, you should see the color change in the app.

3.    Alternatively if you do not see your changes, open the Terminal.
Press Ctrl+C  to cancel the live preview.
Re-run the app in your desktop browser by typing the following command.

ionic serve
